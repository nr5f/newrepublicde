Naval Academy Commander's Exam

(Version 1.01)

Welcome to the commander's exam of the Naval Academy on Coruscant. In the following test 100 points are possible. 60 are nessesary to pass the exam and get the Advanced Training Medal. You can try the test as often as you want to and if you have any questions, feel free to ask any officer in the 5th fleet. The answers for the questions 1 - 8 can be found in the Fleet Manual. To create the html page you can use a text editor or a special html editor if you want to. When you have done the test you can send the answers to High Admiral Matt Sifi (mattsias@gmail.com).
