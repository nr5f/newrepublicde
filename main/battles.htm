TIE Fighter

Here you can download Missions for TIE Fighter. If you have problems to install one of the battles look on the install.txt in each zip file for battles.
Completing 5th Fleet Battle 1 is still credites with awarding the Blue Star. You should start a new pilot file when you start this battle and you should sent this file to you direct commanding officer. (You can find the pilot file in your /TIE or /TIECD directory. The is named the pilot name you used for this battle in TIE Fighter and the file has the extension .TFR)
The other battles are those the 5th fleet used when it was still Imperial. They show a bit the development of the 5th in the first years.
All the battle have been added to the following zip:

battf.zip
