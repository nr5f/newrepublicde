ID Lines


The ID Line is constructed the following way:

Ranking/Position Name/Shiptype Shipname
or if you are member of a Squadron:
Ranking/Position Name/Shiptype Shipname/SquadronName SquadronPosition

optionally Subgroup Positions can be added behind the ID Line

III) Name:
In the Name Slot you can either add your Name or your Callsign.

IV) ShipType and ShipName:
If you are on a ship you should add one of the following terms into the ShipType and Shipname Slots:

ASD Prometheus

ISD Immortal

CRS Payback

VNSD Predator

If you are in cadet in the Naval Academy Coruscant you can add the following:

Naval Academy Coruscant

If you are reserve officer on the Starfortress Luebeck you can add the following:

Starfortress Luebeck

V) SquadronName:
If you are in a Squadron you should add the name of your Squadron into this slot. For example if you are in Red Squadron you should add Red.

VI) SquadronPosition:
Here you should add your Flight Group number, followed by a Minus, followed by the number of the position within your Flight Group. If you are Flight Leader the number for your position is 1, if you are the 1st Flight Member the number is 2, if you are the 2nd Flight Member the number is 3. For example if you are Flight Leader in the 3rd Flight Group you add 3-1 into the slot.
